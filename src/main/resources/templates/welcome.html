HANGMAN GAME

header message

Hey dear player, this funny game has been created by Kane Lo Hog Tian.

The goal of this game is to guess a word randomly generated by the system.

Rules of the Game

  • Each game will generate a random word to be guessed.
  • You have up to 7 incorrect guesses to make.
  • The game ends either when you have guessed the word completely (meaning you win the game) or make more than 7 incorrect guesses (meaning you lose the game).
  • Each time you go back to the welcome page and start a game, a new game with a new random word will be shown.
  • You can also go back to any game by going to localhost:8080/game/id, where id is the game id.
